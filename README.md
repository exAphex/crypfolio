# Crypfolio

Crypfolio is a crypto portfolio tracking app which can calculate tax reports according to german law.

## Functionality

- Import CSV files from different providers
- Load historic price feed from coingecko
- Generate Tax report

## Supported platforms

### Blockchain

- Bitcoin (BTC)

### Exchanges

- Binance
- Crypto.com
- Kraken

### Wallets

- Ledger
